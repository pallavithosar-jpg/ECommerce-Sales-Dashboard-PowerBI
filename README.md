# 📊 E-Commerce Sales Analysis Dashboard

## 📌 Overview
This project is an interactive E-Commerce Sales Dashboard developed using Microsoft Power BI. It provides insights into sales, profit, quantity sold, and category-wise performance using interactive visualizations.

## 🛠 Tools Used
- Microsoft Power BI
- Microsoft Excel
- Data Visualization

## 📈 Dashboard Features
- Total Sales Analysis
- Profit Analysis
- Quantity Sold
- Category-wise Sales
- State-wise Sales
- Interactive Filters (Slicers)
